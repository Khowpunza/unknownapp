# Python Student Menu

This folder contains a Python version of the **student menu** from the Java enrollment system.

## Run

From the project root:

```bash
python3 python/student_menu.py
```

## Features

- View course catalog
- Register for a course
- Drop a course
- View schedule
- View billing summary
- Edit student profile
- Create a new student profile

The script uses the existing JSON files in the repository's `data/` folder.
